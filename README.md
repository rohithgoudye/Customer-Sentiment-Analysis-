\# Customer Sentiment Analysis



\## Overview

A Machine Learning based web application that analyzes customer reviews and predicts sentiment as Positive, Negative, or Neutral.



\## Features

\- Sentiment Prediction

\- Machine Learning Model

\- Interactive Streamlit UI

\- Real-time Text Analysis



\## Technologies Used

\- Python

\- Scikit-learn

\- Pandas

\- Streamlit

\- NLP



\## Model Accuracy

93.92%



\## How to Run



pip install -r requirements.txt



streamlit run app.py
